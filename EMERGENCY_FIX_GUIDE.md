# 🧙‍♂️ Emergency Magic Prompt Guide

Agar aap dusre device par ye project set up karte hain aur wahan **"Failed to Fetch"** wali problem aati he, toh aapko bas Antigravity (mujhe) ye **Magic Sentence** bolna he:

---

### 💬 The Magic Sentence:
> **"Failed to fetch error aa raha he. Please backend/main.py ki CORS settings me Port 3000 aur 5500 ko explicitly allow karo, aur frontend/js/auth.js me API_URL ko absolute IPv4 (127.0.0.1:8000) par set karke stabilize karo."**

---

### ❓ Ye sentence kya karega?
1.  **Direct Instructions**: Ise sunte hi Antigravity samajh jayega ki use browser security (CORS) aur Windows networking issues ko fix karna he.
2.  **No Research Needed**: Mujhse research karne me time waste nahi hoga, main turant files edit karke connection fix kar dunga.
3.  **Port Sync**: Ye use batayega ki aapka launcher Port 3000 use karta he, isliye wo port allow hona zaroori he.

### ✅ Quick Self-Check Check-list:
Agar error aaye, toh pehle ye 2 cheezein zaroor check karein:
1.  Kya `py run_demo.py` chala rakha he? (Yaani backend door open he?)
2.  Kya browser me **Ctrl + F5** (Hard Refresh) kiya he? (Yaani purana cache clear he?)

Ye guide aapke project folder me hamesha rahegi! 🚀🎓✅
